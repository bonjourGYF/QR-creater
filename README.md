# 挪车联系二维码

这是一个用于生成“挪车联系卡片二维码”的静态网页。二维码内容是网页链接，微信扫码后会打开联系卡片页面，显示车主电话、微信号、车牌号和提示语。

## 在线使用

打开网页：

https://bonjourgyf.github.io/QR-creater/

在页面中填写需要展示的信息：

- 标题，例如“扫码联系车主挪车”
- 车主称呼，例如“车主”
- 电话号码
- 微信号
- 车牌号
- 提示语，例如“您好，如需挪车请联系我，谢谢。”

电话和微信至少填写一种，否则无法生成可联系的二维码。

## 生成二维码

填写信息后，点击“生成二维码”。

页面会生成一个网页链接二维码。这个二维码不是纯文本二维码，而是类似下面这样的链接：

```text
https://bonjourgyf.github.io/QR-creater/?mode=card&title=...&phone=...&wechat=...&plate=...&message=...
```

微信扫码后会打开一个适合手机查看的挪车联系卡片页面。

## 下载和打印

生成二维码后，可以使用页面上的按钮：

- “下载二维码 PNG”：保存清晰的二维码图片，适合打印
- “打印挪车卡片”：直接打印包含二维码的挪车卡片
- “复制生成的网页链接”：复制当前生成的联系卡片链接

打印前建议先用手机微信扫码测试一次，确认页面能正常打开，电话和微信信息正确。

## 联系卡片功能

扫码打开的联系卡片页面包含：

- 车主电话
- 微信号
- 车牌号
- 提示语
- 一键拨打电话按钮
- 复制微信号按钮

## 注意事项

- 建议只填写电话或微信，不要填写住址、身份证等敏感信息。
- 如果以后电话、微信或车牌号变了，需要重新生成二维码并重新打印。
- 只要仓库和 GitHub Pages 地址不变，已经生成并打印的二维码可以长期使用。
